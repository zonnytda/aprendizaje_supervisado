Descripción del Proyecto
Predicción de Promedio Final de Egresados UNE

Sistema de Machine Learning supervisado para predecir el promedio final de egresados de la Universidad Nacional de Educación Enrique Guzmán y Valle (La Cantuta).

Este proyecto implementa algoritmos de regresión regularizada (Ridge y Lasso) para analizar y predecir el rendimiento académico de estudiantes egresados, utilizando características demográficas, académicas e institucionales.

Algoritmos de Machine Learning
Ridge Regression

Regresión lineal con regularización L2 que penaliza la magnitud de los coeficientes, ayudando a prevenir el overfitting.

Ventajas:

  • Maneja bien la multicolinealidad
  • Estable y robusto
  • Buena interpretabilidad
Lasso Regression

Regresión lineal con regularización L1 que puede reducir algunos coeficientes a cero, realizando selección automática de features.

Ventajas:

  • Selección automática de features
  • Genera modelos sparse
  • Buena para datos de alta dimensionalidad
¿Por qué estos algoritmos?

Ridge y Lasso son ideales para este problema porque:

  • Manejan bien datasets con muchas features categóricas (one-hot encoding)
  • Previenen overfitting mediante regularización
  • Tienen alta interpretabilidad (importante en contexto educativo)
  • Son computacionalmente eficientes

Metodología
Pipeline de Machine Learning
  1. Carga de Datos: Lectura del dataset desde CSV
  2. Preprocesamiento:
    • Identificación de tipos de variables
    • Manejo de valores nulos (estrategia: drop)
    • Detección y eliminación de outliers (método IQR)
    • División estratificada train/val/test (70/15/15)
  3. Feature Engineering:
    • Creación de 7 features derivadas
    • One-Hot Encoding para categóricas
    • Standard Scaling para numéricas
    • Agrupación de categorías raras
  4. Entrenamiento:
    • Grid Search para optimización de hiperparámetros
    • Validación cruzada (k=5)
    • Entrenamiento de Ridge y Lasso
  5. Evaluación:
    • Métricas: RMSE, MAE, R², MAPE
    • Análisis de residuales
    • Learning curves
    • Feature importance
  6. Comparación: Selección del mejor modelo
  7. Despliegue: Aplicación web interactiva
Métricas de Evaluación
Métrica Descripción Interpretación
RMSE Root Mean Squared Error Error promedio en la misma escala que la variable objetivo. Más bajo es mejor.
MAE Mean Absolute Error Error absoluto promedio. Más robusto a outliers que RMSE.
R² Coeficiente de Determinación Proporción de varianza explicada (0-1). Más alto es mejor.
MAPE Mean Absolute Percentage Error Error porcentual promedio. Útil para comparar modelos.
